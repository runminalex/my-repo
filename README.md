# my-repo

A simple "Hello, World!" website served via Python's built-in HTTP server.

## Description

This is the first project for `my-repo` — a minimal static website that displays a greeting. It serves as the starting point for the multi-agent GitOps pipeline.

## Prerequisites

- **Python 3** (stdlib only — no external dependencies)

## Quick Start

bash
chmod +x run.sh
./run.sh


Then open [http://127.0.0.1:8080](http://127.0.0.1:8080) in your browser.
